# Dungeon Eval Toolkit

A web-based Dungeon Crawler puzzle environment for evaluating AI language models on spatial reasoning and planning tasks.

## Quick Start

```sh
bun install
bun dev
```

Open http://localhost:5173 to play or build levels.
